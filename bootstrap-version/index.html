Sobre mim

Sistemas simples
que funcionam de verdade

Olá, sou Gilmar Filho, estudante de Sistemas de Informação na UFLA (6º período). Trabalho construindo automações para quem procura ganhar tempo delegando a um sistema suas tarefas repetitivas.

Minha especialidade é transformar processos manuais em fluxos automáticos — usando Google Planilhas com Apps Script, Python e integrações com ferramentas que as pessoas já usam no dia a dia, como o Telegram.

O que guia cada projeto é uma escolha deliberada: entregar o que resolve com qualidade, não o que impressiona com quantidade. Um sistema com 3 funcionalidades que o usuário usa todos os dias vale mais do que um sistema "completo" que no fim vira um peso.

Gestão de Estoque via Telegram

Google Planilhas · Webhook · Telegram Bot

Loja de materiais de construção gerenciada "de cabeça". A gerente tem pouco tempo e baixa tolerância para sistemas com fluxos longos.

Solução sem interface: Google Planilhas como base de dados invisível ao usuário — integrada a um bot no Telegram. Cadastro, entradas, saídas e alertas de reposição pelo app que a cliente já usa. Zero tela de sistema, zero treinamento. Tudo adaptado ao fluxo diário da cliente.
