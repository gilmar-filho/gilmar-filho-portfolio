# Gilmar Filho | Portfólio Profissional

Portfólio pessoal desenvolvido para apresentar minhas habilidades em Engenharia de Software, Automação e Sistemas Distribuídos. O projeto foi construído com foco em responsividade, UI/UX moderno e boas práticas de desenvolvimento web.

## 🏗️ Arquitetura do Projeto

Este repositório atende a um requisito acadêmico (disciplina de Programação Web - UFLA) de demonstrar proficiência em diferentes ecossistemas CSS. Por isso, o projeto é dividido em duas versões espelhadas:

* **Versão Principal (Raiz):** Desenvolvida com **Tailwind CSS v3**, apresentando um design altamente customizado, efeitos visuais avançados (Glows, Backdrop Blur) e um terminal interativo construído do zero.
* **Versão Acadêmica (`/bootstrap-version/`):** Desenvolvida utilizando os componentes nativos e o sistema de grid do **Bootstrap 5.3**, mantendo a mesma semântica e hierarquia de informações.

## 🛠️ Tecnologias Utilizadas

* **Linguagens:** HTML5, CSS3, JavaScript
* **Frameworks/Bibliotecas CSS:** Tailwind CSS v3, Bootstrap 5.3
* **Ícones e Tipografia:** Phosphor Icons, Google Fonts (Inter, Monospace)
* **Funcionalidades Globais:** * Alternância de Tema (Light/Dark Mode) interligada com `localStorage` e preferências do sistema.
    * Manipulação de DOM para modais dinâmicos com efeito de desfoque (blur) no background.

## 💼 Projetos Destacados no Portfólio

O portfólio documenta meus principais trabalhos e experimentos:

1.  **NiadAgent:** Automação de transcrição e sumarização de reuniões (Docker, Tailscale, Whisper, Telegram API, Gemini API).
